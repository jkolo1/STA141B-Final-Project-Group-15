Housing Affordability Across U.S. Metro Regions 

Overview:
This project examines how the drastic increase in home values and the impact the monetary influxes have on housing affordability across U.S. metropolitan areas. As home prices have significantly risen over the past few decades, the gap between what homes cost and what households can afford has widened. This analysis focuses on translating complex housing data into clear and comprehensible information that is accessible. This can be beneficial for first time homebuyers who may be trying to understand the housing market. Using publicly avaiable and constantly updated data from Zillow, a Price-to-Income Ratio has been constructed to track the trends from 2012-2026. 

Datasets: 
Both datasets are sourced from Zillow Research (https://www.zillow.com/research/data/).
- Metro_zhvi_uc_sfrcondo_tier_0.33_0.67_sm_sa_month.csv — Zillow Home Value Index (ZHVI), contains monthly home values and is organized by regions, spanning 2000–2026
- Metro Homeowner Income Downpayment Data.csv — New Homeowner Income Needed, contains the estimated annual income required to afford homeownership, spanning 2012-2026

How to Run:
1. Clone the repository
2. Open Housing Project.ipynb in Jupyter Notebook
3. Run all Cells in order

Files in Repo:
- Housing Project.ipynb: main analysis notebook, includes data cleaning.
